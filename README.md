# ChivanCOM Kodi Repository

Install the ChivanCOM repository in Kodi to get easy access to all ChivanCOM add-ons, including the **iBroadcast Music** plugin.

## Install via zip file (recommended)

1. Download the latest repository zip:
   **[plugin.repository.chivancom-1.0.9.zip](https://chivancom.github.io/kodi-repository-install/plugin.repository.chivancom-1.0.9.zip)**

2. In Kodi, go to **Add-ons** → **Install from zip file**.

3. Navigate to the downloaded zip and select it.

4. Kodi will install the ChivanCOM repository. You can now browse and install any ChivanCOM add-on from **Add-ons** → **Install from repository** → **ChivanCOM Repository**.

> **Note:** If Kodi asks about unknown sources, go to **Settings** → **System** → **Add-ons** and enable **Unknown sources** first.

---

## Install via repository URL (advanced)

You can also point Kodi directly at this repository URL so it always fetches the latest version automatically.

1. In Kodi, go to **Settings** → **File manager** → **Add source**.

2. Enter the repository URL:
   ```
   https://chivancom.github.io/kodi-repository-install/
   ```
   Give it a name, e.g. `ChivanCOM`.

3. Go to **Add-ons** → **Install from zip file**, select the **ChivanCOM** source, and install the zip from there.

4. Once installed, Kodi will keep the repository up to date automatically.
